# Desenvolvimento


Para utilizar o sisriso faz-se necessário configurar algumas variáveis de ambiente. Crie o arquivo `.env` com os valores apropriados, utilizando o arquivo `.env.exemplo` como base:

    cp .env.exemplo .env
